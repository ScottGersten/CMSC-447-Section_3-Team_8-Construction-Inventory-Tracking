# CMSC-447-Section_3-Team_8-Construction-Inventory-Tracking

## React Installs
### JavaScript
- https://nodejs.org/en
- npx create-expo-app@latest ConstructionInventory
- npm install
### Firebase
- npx expo install firebase
### Run
- npx expo start

## Flutter
- Install Flutter SDK
    - Can be done by installing Flutter and Dart extensions in VS Code, which will prompt with installation
- Flutter run
